# Foundry Lancer Narrative Check Macro

Our group has decided to overhaul Lancer narrative rules with a homebrew set similar to Blades in the Dark or the current Far Field Team draft.

## Usage

Create a new macro in Foundry, set to "script" type, and paste in the contents of main.ts

## Features

- dialog that queries for user-inputted params, and determines the number of dice to roll
    - a variety of form fields populated with options such as the player's active character's skills, burdens, and gear
    - stripped-down version for gm use
- chat message displaying dice results and outcome
